# Dashboard de Alunos

Tela de gerenciamento de alunos desenvolvida em HTML, CSS e JavaScript.

## Funcionalidades

- Listagem de alunos em tabela
- Busca por nome ou telefone
- Filtro por status (Ativo/Inativo)
- Cadastro de novo aluno
- Edição de aluno existente
- Exclusão de aluno
